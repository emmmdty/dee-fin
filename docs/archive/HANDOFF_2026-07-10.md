## 3. 新方案：三章 + 三个算法创新点

**主线**：通用事件图谱的**可验证**自动化构建与应用。
**贯穿**：同一验证器内核的**三重身份** —— 推理门控 · 训练奖励 · 风险控制器。

### ch1 · 事件抽取（节点）
schema 接地的生成式事件抽取（触发词 + 论元）。主表 **MAVEN / MAVEN-ARG**；
验证域 ChFinAnn 86.0 / DuEE-Fin 78.0（SARGE 收编重做）。
口径借 Text2KGBench 的 ontology conformance / hallucination。

### ch2 · 事件图谱自动化构建（边）
LLM 事件关系抽取 + grounding 验证器 + 一致性仲裁。主表 **MAVEN-ERE**；验证域 CCKS-2021 中文因果。
agent 味：proposer / grounding-verifier / consistency-arbiter（对标 KARMA，但我们有保证）。

- **★A1｜事理结构约束作为可验证奖励（GRPO-RLVR）**
  事件图谱天然带可自动验证的结构约束：**时序传递闭包、因果无环、共指等价类、子事件层级**。
  做成 verifier → 复合可验证奖励（format + grounding + **structure** + gold F1）→ GRPO 训练。
  *新颖性*：DAO 是 tuning-free；VeriGate(2605.30451) / Faithful GRPO(2604.08476) 是通用 verifier-gated GRPO；
  **把事理结构当 RLVR 奖励，检索未见先例**（终稿前须复核这两篇的具体做法）。
- **★A2（构建侧）｜CRC 边准入**：FNR ≤ α_e 的有限样本召回保证。

### ch3 · 应用 —— **CGEP**，基座 SeDGPL

三个"小修补"，各挂在 SeDGPL 的一个模块上：

- **M1｜风险感知的图线性化（Risk-aware DsGL）** ★ 直接消解它自陈的 limitation
  它按最短路距离启发式截断三元组。我们把 **CRC 边准入分数**引入截断准则：
  token 预算下选边，使 **FNR ≤ α_e** 有限样本成立。工程妥协 → 有保证的算法组件。
- **M2｜结构感知的因果编码（Structure-aware EeCE）**
  它融合 graph / contextual / schema 三路。我们加**第四路结构一致性特征**
  （候选是否引入因果环、是否违反时序传递闭包、共指等价类归属）作为门控信号；
  同一验证器同时充当 A1 的 GRPO 奖励。
- **M3｜选择性对比预测（Selective ScEP → CS-CRP head）**
  它的 ScEP 确定性输出单个事件。换成 **conformal 预测集 + 弃权**。

- **★A2（应用侧）｜CS-CRP：跨阶段 conformal 风险传播**
  构建期 FNR ≤ α_e ⊗ 应用期漂移鲁棒覆盖 1−α_p，显式建模「**上游漏边使 gold 后继事件不可达**」
  的覆盖↔可达性权衡。*新颖性*：DAO 单阶段、PASC 假设可交换且同质、2605.05476 只经验量化、MDPI 自陈没有。
- **★A3｜赋能 LLM**：把验证过的因果图 + CS-CRP 作为结构先验喂给 LLM predictor，
  论证「可验证构建 + 形式化保证 → LLM 从落后于专用模型变成带保证的选择性预测器」。
  EventRAG 那条 RAG 线降级为相关工作。
- **杀手实验**：**gold ECG vs 我们构建的 ECG** 上的 CGEP 性能差 = 误差传播曲线；CS-CRP 给它一个界。

**竞争策略**：不在 MRR 上硬拼。gold 图上追平 SeDGPL 即可；主张落在
*constructed 图上的鲁棒性* + *selective accuracy@coverage / AURC / 组合覆盖保证*。
